
# Занятие 10: Доработка индивидуальных проектов и демонстрация

## Цель занятия
Завершить оба проекта (VR-Тир и AR-Тир), добавить финальные улучшения и подготовить демонстрацию.

### Шаг 1: Объединение VR и AR в одном проекте
1. Создайте пустую сцену `MainMenu`.
2. Создайте Canvas с двумя большими кнопками:
   - "Запустить VR Тир"
   - "Запустить AR Тир"
3. На кнопки назначьте скрипт, который загружает нужную сцену:

```csharp
using UnityEngine;
using UnityEngine.SceneManagement;

public class MenuManager : MonoBehaviour
{
    public void StartVR()
    {
        SceneManager.LoadScene("VR_Tir_Main");
    }

    public void StartAR()
    {
        SceneManager.LoadScene("AR_Tir_Main");
    }
}
```

### Шаг 2: Финальные улучшения
1. Добавьте в оба проекта:
   - Главное меню с названием игры.
   - Кнопку "Выход".
   - Таблицу лучших результатов (простой список).
2. Оптимизируйте производительность (уменьшите полигоны, отключите лишние эффекты).
3. Добавьте финальные эффекты (конфетти при высоком счёте).

### Шаг 3: Подготовка к демонстрации
1. Сделайте финальные Build'ы под Android (отдельные apk для VR и AR).
2. Подготовьте презентацию:
   - Как работает VR версия.
   - Как работает AR версия.
   - Что вы научились делать.
3. Запишите короткое видео демонстрации (30–60 секунд).

### Шаг 4: Рефлексия
Обсудите в группе:
- Что получилось лучше всего?
- Что было самым сложным?
- Что бы вы добавили в игру дальше?
