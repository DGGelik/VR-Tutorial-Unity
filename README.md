# Курс “AR и VR приложения в Unity + Нейросети”

**Игра-проект: Тир (VR + AR)**

Добро пожаловать! Здесь собраны пошаговые туториалы для самостоятельной работы дома.

## Структура курса
- **Занятие 1–5**: Создаём **VR-Тир**
- **Занятие 6–10**: Создаём **AR-Тир**

## Как работать
1. Скачай репозиторий.
2. Открой Unity Hub → Add project from disk.
3. Следуй туториалам по номерам занятий.
4. Делай build под Android и тестируй на телефоне.
