# Занятие 2: Создание 3D-локации тира с ProBuilder

## Что делаем
Создаём окружение виртуального тира.

## Пошагово

1. **ProBuilder**
   - Window → Package Manager → Установи ProBuilder.
   - Создай стены, стойку для стрельбы, барьеры.

2. **Мишени**
   - Создай 5–8 разных мишеней (Cube, Cylinder).
   - Размести их на разных расстояниях.

3. **Asset Store**
   - Найди и импортируй бесплатные ассеты:
     - Low Poly Guns
     - Particle Effects
     - Simple Targets

4. **Skybox 360°**
   - Скачай или сгенерируй (Kandinsky) текстуру интерьера тира.
   - Примени как Skybox.

5. **Освещение**
   - Добавь Directional Light + несколько Point Lights.

## Результат
Готовая визуальная локация тира.

**Домашнее задание:** Доработай комнату, сделай красивой.
