# Занятие 4: Создание прототипа 3D-мира. Совершенствование в ProBuilder. Материалы и текстуры

## Цель занятия
Сделать локацию тира красивой и профессиональной. Мы будем очень подробно работать с ProBuilder, создавать материалы и настраивать текстуры шаг за шагом.

### Шаг 1: Подготовка ProBuilder
1. Открой сцену `VR_Tir_Main`.
2. В верхнем меню выбери **Tools** → **ProBuilder** → **ProBuilder Window**.
3. Если окно маленькое — перетащи его правый край, чтобы было удобно.
4. В ProBuilder Window внизу нажми на иконку **Toolbar**, чтобы были видны все кнопки.

### Шаг 2: Улучшение стен и пола (очень подробно)
1. В Hierarchy выдели Plane (пол).
2. В Inspector поставь Scale: `X=25`, `Y=1`, `Z=25`.
3. Выдели одну стену.
4. В ProBuilder Window нажми кнопку **Face** (выбор граней).
5. Кликни на нижнюю грань стены.
6. Нажми кнопку **Extrude** (выдавить) и потяни мышкой вниз на 0.3 метра — получится плинтус.
7. Кликни на верхнюю грань стены → **Extrude** вверх на 0.5 метра — сделай карниз.
8. Повтори эти действия для всех четырёх стен.
9. Для стойки стрельбы:
   - Создай новый Cube (**GameObject** → **3D Object** → **Cube**).
   - Position: `X=0, Y=0.8, Z=3`.
   - Scale: `X=8, Y=0.1, Z=1.5`.
   - Выдели его в ProBuilder → нажми **Bevel** → введи значение 0.08 → Apply.

### Шаг 3: Создание материалов (пошагово)
1. В Project окне зайди в Assets.
2. Правой кнопкой → **Create** → **Folder** → назови папку `Materials`.
3. Внутри папки Materials правой кнопкой → **Create** → **Material**.
4. Назови материал `Floor_Concrete`.
5. Выдели этот материал → в Inspector:
   - Цвет Albedo — выбери светло-серый (#A8A8A8).
   - Smoothness — поставь значение 0.25.
   - Metallic — поставь 0.1.
6. Перетащи материал `Floor_Concrete` на Plane в сцене.
7. Создай материал `Wall_Metal`:
   - Albedo — тёмно-серый.
   - Smoothness — 0.75.
   - Metallic — 0.9.
8. Примени его ко всем стенам.

### Шаг 4: Добавление и настройка текстур
1. Создай папку `Textures` в Assets.
2. Перетащи туда текстуры бетона и металла (если нет — используй цвета пока).
3. Выдели материал `Floor_Concrete`.
4. В поле **Albedo** перетащи текстуру бетона из папки Textures.
5. В полях **Tiling** X и Y поставь значение `8` (чтобы текстура повторялась чаще).
6. То же самое сделай для стен.

### Шаг 5: Освещение комнаты
1. Выдели Directional Light в Hierarchy.
2. В Inspector:
   - Intensity = 1.3
   - Color — слегка оранжевый оттенок.
3. Создай 6 Point Light:
   - **GameObject** → **Light** → **Point Light**.
   - Расставь их под потолком по углам и над стойкой.
   - Для каждой лампы Intensity = 2.5, Range = 10.

### Шаг 6: Финальная проверка
1. Нажми кнопку **Play**.
2. Осмотри комнату со всех сторон.
3. Сделай Build And Run.
4. Протестируй в VR Box.

**Что должно быть готово к концу урока:**
- Красивый пол и стены с материалами.
- Хорошее освещение.
- Много мишеней.
- Комната выглядит как настоящий тир.
