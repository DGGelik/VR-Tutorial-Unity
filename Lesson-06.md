# Занятие 6: Введение в технологию дополненной реальности. Vuforia Engine AR. Прототип AR-проекта

## Цель занятия
Перейти от VR к AR. Научиться использовать Vuforia, создать первый прототип AR-Тира и собрать его под Android.

### Шаг 1: Установка Vuforia
1. Откройте **Window** → **Package Manager**.
2. В левом верхнем углу переключите на **Unity Registry**.
3. В поиске напишите **Vuforia**.
4. Найдите **Vuforia Engine AR** и нажмите **Install**.

### Шаг 2: Создание новой AR-сцены
1. В папке Assets правой кнопкой → **Create** → **Scene**.
2. Назовите сцену `AR_Tir_Main`.
3. Дважды кликните, чтобы открыть её.

### Шаг 3: Настройка AR-камеры
1. Удалите Main Camera из Hierarchy.
2. В верхнем меню **GameObject** → **Vuforia Engine** → **AR Camera**.
3. Вы увидите AR Camera в Hierarchy — это специальная камера для дополненной реальности.

### Шаг 4: Создание Image Target (метки)
1. **GameObject** → **Vuforia Engine** → **Image Target**.
2. Выделите Image Target в Hierarchy.
3. В Inspector в поле **Type** выберите **Predefined**.
4. Нажмите кнопку **Open Vuforia Configuration** (или перейдите на сайт developer.vuforia.com).
5. Зарегистрируйтесь на сайте https://developer.vuforia.com/ (если ещё не делали).
6. Создайте новый License Key и Database.
7. Скачайте Database и импортируйте в Unity (двойной клик по .unitypackage).
8. В Image Target выберите вашу метку из базы данных.

### Шаг 5: Перенос объектов из VR в AR
1. Откройте сцену VR_Tir_Main и скопируйте мишени, стойку и спавнер.
2. Вставьте их в сцену AR_Tir_Main.
3. Сделайте все объекты детьми (Child) Image Target (перетащите в Hierarchy на Image Target).
4. Уменьшите масштаб, если нужно (обычно 0.1–0.3).

### Шаг 6: Настройка и сборка
1. **File** → **Build Settings** → Android → Switch Platform.
2. В Player Settings → XR Plug-in Management → включите Vuforia.
3. **Build And Run**.

### Шаг 7: Тестирование
1. Распечатайте или откройте на другом экране Image Target.
2. Запустите приложение на телефоне.
3. Наведите камеру на метку — мишени должны появиться в реальном мире.

**Что должно работать:**
- Мишени появляются поверх реальной картинки-метки.
- Вы можете стрелять в них (используя контроллер или тач).
- Работает система очков.

**Результат занятия:** Первый рабочий прототип **AR-Тира**.
