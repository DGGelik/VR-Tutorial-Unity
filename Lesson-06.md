# Занятие 6: Переход в AR с Vuforia

## Пошагово

1. Установи Vuforia Engine через Package Manager.
2. Создай новую сцену `AR_Tir_Main`.
3. Добавь ARCamera + Image Target.
4. Перенеси мишени и оружие из VR-проекта.
5. Размести объекты как children Image Target.
6. Настрой лицензию Vuforia (Developer Portal).
7. Сделай Build и протестируй на реальной метке.

**Результат:** Мишени появляются в реальном мире.
