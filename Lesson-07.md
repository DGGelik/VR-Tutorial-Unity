# Занятие 7: Создание баз данных и ключей для AR-игр. Основы работы со звуком

## Цель занятия
Научиться создавать и использовать несколько Image Targets, добавить звук в AR-Тир.

### Шаг 1: Работа с Vuforia Developer Portal
1. Откройте браузер и зайдите на сайт https://developer.vuforia.com/.
2. Войдите в аккаунт.
3. В меню выберите **Target Manager**.
4. Нажмите кнопку **Add Database** → назовите её `TirDatabase`.
5. Откройте базу данных и нажмите **Add Target**.
6. Выберите тип **Single Image**.
7. Загрузите 3–4 чёткие картинки (логотипы, фото, узоры) — они будут вашими метками.
8. Для каждой метки поставьте Width = 0.2 и нажмите **Submit**.
9. Когда статус станет **Active**, скачайте базу данных (кнопка Download).

### Шаг 2: Импорт базы в Unity
1. Перетащите скачанный .unitypackage файл в Unity (в Project окно).
2. Unity предложит импортировать — нажмите **Import**.
3. В Hierarchy выберите Image Target.
4. В Inspector в поле **Database** выберите вашу новую базу `TirDatabase`.
5. В поле **Image Target** выберите одну из загруженных меток.

### Шаг 3: Создание нескольких меток
1. Создайте 3–4 объекта Image Target.
2. Для каждого укажите свою метку из базы.
3. На каждый Image Target перенесите разные наборы мишеней (например, одна метка — лёгкий тир, другая — сложный).

### Шаг 4: Добавление звука
1. Скачайте или найдите звуки:
   - Выстрел.
   - Попадание.
   - Фоновая музыка.
2. Перетащите звуковые файлы (.wav или .mp3) в папку Assets → `Sounds`.
3. На объект Gun добавьте компонент **Audio Source**.
4. В поле **Audio Clip** перетащите звук выстрела.
5. Поставьте галочку **Play On Awake** — снимите.
6. В скрипте `SimpleGun.cs` убедитесь, что есть строка `shotSound.Play();`.

### Шаг 5: Настройка Audio Listener
1. На AR Camera должен быть компонент **Audio Listener** (обычно есть по умолчанию).
2. Если нет — добавьте (Add Component → Audio Listener).

### Шаг 6: Сборка и тестирование
1. **File** → **Build Settings** → Android.
2. **Build And Run**.
3. Распечатайте несколько меток.
4. Протестируйте: наведите камеру на разные метки — должны появляться разные мишени + работать звук.

**Результат занятия:** AR-Тир с несколькими метками и звуком.
