

# Занятие 1: Введение в Unity + Первый прототип VR-Тира

## Цель занятия
Познакомиться с Unity и создать самый первый рабочий прототип тира, который можно будет запустить на телефоне через VR Box.

### Шаг 1: Создание проекта
1. Открой **Unity Hub**.
2. Нажми большую синюю кнопку **New Project** (справа вверху).
3. Выбери шаблон **3D (URP)** — он оптимизирован для мобильных устройств и AR/VR.
4. В поле **Project name** напиши: `Unity-Tir-Course`
5. Нажми **Create Project**. Подожди, пока проект полностью загрузится (может занять 1–2 минуты).

**Зачем мы это делаем?** Каждый проект Unity — это отдельная папка на компьютере, где будут храниться все файлы игры.

### Шаг 2: Знакомство с интерфейсом
После загрузки обрати внимание на основные окна:
- **Hierarchy** (слева) — иерархия всех объектов сцены.
- **Scene** (центр) — окно редактирования мира.
- **Game** (рядом) — как будет выглядеть игра для игрока.
- **Inspector** (справа) — свойства выбранного объекта.
- **Project** (внизу) — все файлы проекта (папка Assets — главное место работы).

### Шаг 3: Очистка и создание новой сцены
1. В окне **Project** (внизу) перейди в папку `Assets`.
2. Правой кнопкой мыши → **Create** → **Scene**. Назови её `VR_Tir_Main`.
3. Дважды кликни по `VR_Tir_Main`, чтобы открыть сцену.

**Почему удаляем стандартную сцену и объекты?**  
Стандартная сцена содержит примеры, которые нам не нужны. Мы создаём чистую сцену специально под наш тир, чтобы ничего не мешало.

4. В **Hierarchy** удали объекты **Main Camera** и **Directional Light** (правой кнопкой мыши → Delete).  
   **Зачем?** Мы будем использовать специальную камеру для VR.

### Шаг 4: Создание первого объекта (мишень)
1. В верхнем меню: **GameObject** → **3D Object** → **Plane** (это будет пол).
2. Выдели Plane в Hierarchy → в Inspector измени **Scale**: `X=10`, `Y=1`, `Z=10`.
3. Создай мишень: **GameObject** → **3D Object** → **Cube**.
4. Выдели Cube и в Inspector установи:
   - Position: `X=0, Y=1.2, Z=6`
   - Scale: `X=1, Y=1.8, Z=0.3`
5. Создай материал: в Project → правой кнопкой → **Create** → **Material**. Назови `RedTarget`.
6. В Inspector материала выбери ярко-красный цвет и перетащи материал на Cube.

### Шаг 5: Настройка VR для VR Box
1. В верхнем меню: **Edit** → **Project Settings** → **XR Plug-in Management**.
2. В разделе **Android** поставь галочку напротив **OpenXR**.
3. Если OpenXR не установлен — Unity предложит скачать. Установи.
4. В **OpenXR** настройках выбери **Google Cardboard** (или **Cardboard**) как основной профиль.

**Пояснение:**
- **OpenXR** — современный стандарт для VR/AR.
- **Google Cardboard** — позволяет использовать гироскоп телефона для поворота головы в игре.
- VR Box вставляет телефон — поэтому мы полагаемся на гироскоп телефона.

### Шаг 6: Настройка управления (стрельба и движение)
1. Создай пустой объект: **GameObject** → **Create Empty**. Назови `Player`.
2. Добавь на него компонент **Camera** (Add Component → Camera).
3. Создай скрипт `SimpleGun.cs` (в Project → Create → C# Script).

```csharp
using UnityEngine;

public class SimpleGun : MonoBehaviour
{
    public float range = 100f;

    void Update()
    {
        // Стрельба по нажатию кнопки мыши или контроллера
        if (Input.GetButtonDown("Fire1") || Input.GetKeyDown(KeyCode.JoystickButton0))
        {
            Shoot();
        }
    }

    void Shoot()
    {
        Ray ray = new Ray(transform.position, transform.forward);
        if (Physics.Raycast(ray, out RaycastHit hit, range))
        {
            Debug.Log("Попадание в: " + hit.transform.name);
            // Позже здесь будет уничтожение мишени
        }
    }
}
```

4. Перетащи скрипт на объект Camera.

### Шаг 7: Первый Build
**File** → **Build Settings** → Android → **Switch Platform** → **Build And Run**.

**Тестирование:** Вставь телефон в VR Box, подключи Bluetooth-контроллер. Поворачивай голову (гироскоп телефона), стреляй кнопкой на контроллере.
